终于，这个站算是活过来了。

说实话，手里握着华为云新加坡、甲骨文的 ARM 机子，还有刚抽奖中的搬瓦工 MegaBox，如果不搭点什么东西，总觉得对不起这些 VPS 的电费。但真要动工的时候，我又不想用 WordPress（太重）或者 Hexo（环境配置太烦）。

我想搞点不一样的——**极致轻量，纯手搓，还得够 Geek**。

### 审美强迫症的开局

最开始，我只是想要一个“极简”的页面。

原本的字体是宋体，看着太锐利，像是在看打印出来的红头文件。不行，得改。我花了一下午调试 CSS，把字体全换成了 Nunito 配合圆体（幼圆/雅黑UI），看着那个圆润的英文字母，心里才舒服了。

还有标题那个打字机光标 `_`，必须得让它闪起来，这才是 Linux 终端该有的感觉。文案从倒计时改成“花有重开日，人无再少年”，瞬间那个味儿就对了。

### 与 Docker 和 SQLite 的“血泪史”

原本想着既然有 VPS，评论系统 Waline 肯定要自己部在 Docker 里，数据握在自己手里才安心。

结果，**心态崩了**。

用 1Panel 部署的时候，挂载路径把我搞晕了。一会是 `SQLITE_ERROR: no such table`，一会是容器启动了但数据写不进去。明明教程看了好几遍，就是因为一个路径没对上，或者权限不够（Linux 的权限管理有时候真让人头大），折腾了大半宿。

最后看着那堆红色的 Error Log，我悟了：**能白嫖为什么要自己运维？**

果断转投 Vercel + Neon。不用管 Docker，不用管端口映射，一键部署，真香。

### 与“403 Forbidden”的斗争

功能搞定后，我拿 ITDog 测了一下速，**全国平均 1.9 秒**。

看到这个数字我差点把键盘砸了。打开 F12 一看，好家伙，`npm.elemecdn.com` 全线飘红，报 403 Forbidden。饿了么的 CDN 应该是开了防盗链或者挂了，导致评论区直接消失，Marked 解析库也加载巨慢。

没办法，只能进行“外科手术式”换源。
把 JS 库换到了 360 的 Baomitu，Waline 换回了 JSDelivr，顺便把那个拖慢速度的 Google 在线字体砍了，直接调用系统本地字体。

现在再测，秒开。舒坦。

### 最后的“亿”点点细节

为了让这个博客更像是一个“家”，我还手搓了一个类似 QQ 空间/朋友圈的“动态”功能。不用发文章那么正式，以后有什么碎碎念，直接写个标签就能以卡片流的形式展示出来。

最后把底部的版权改成了 `Cloud Computing Support`，挂上了 GitHub 和 Cloudflare 的标（虽然是静态站，但逼格不能丢）。

**折腾，可能就是作为iKun/MJJ的浪漫吧。**

2026，你好。
