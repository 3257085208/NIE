# 🌸 Higan-Static (聶.NET 博客系统)

这是一个基于 **GitHub Pages + GitHub Actions** 的超轻量级静态博客系统。
无需服务器（除非你需要评论后端），无需复杂的 Hexo/Hugo 构建环境，**仅需一个 HTML 文件**即可运行。

界面完美复刻 Halo Higan 主题，支持极致的移动端适配和暗黑模式。

## ✨ 特性

* **极简架构**：核心仅由 `index.html` (前端) 和 `build.yml` (构建脚本) 组成。
* **自动化构建**：写好 Markdown 文件推送到 GitHub，Actions 自动生成文章索引和字数统计。
* **Higan 复刻**：1:1 还原 Higan 主题的经典布局（PC 端纯文字导航，移动端侧滑菜单）。
* **功能全**：
    * 🌑 丝滑的日夜模式切换
    * 💬 Waline 评论系统集成
    * 📊 真实字数统计 & 不蒜子访问量统计
    * 🏷️ 支持自定义分类 & 标签 (Front Matter)
    * 📱 完美的移动端手势适配 & 悬浮按钮

---

## ✍️ 如何写文章 (Usage)

你可以在电脑上，也可以直接用手机 GitHub App 上传文件，**无需任何本地环境**。

### 1. 创建文件
在仓库的 `posts` 文件夹下新建一个 `.md` 文件。
**文件名建议格式**：`YYYY-MM-DD-文章标题.md` (例如: `2026-01-21-你好世界.md`)

### 2. 文章格式 (Front Matter)
在文章的最开头，**必须**加上如下的配置块（YAML 格式），否则系统会自动使用文件名作为标题，分类默认为“默认”。

```yaml
---
title: 这里写文章标题
date: 2026-01-21
category: 日常吐槽
tags: [VPS, Linux, 生活]
---

这里开始写正文...
支持 **Markdown** 语法。
